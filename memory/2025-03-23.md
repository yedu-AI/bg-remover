#### 飞书生态 (4个)
- **feishu-doc**: 飞书文档操作 (读写、创建、表格、图片上传)
- **feishu-drive**: 飞书云存储管理 (列出、移动、删除)

- **coding-agent**: 编码代理委托 (Codex/Claude Code/Pi，后台进程管理)

- **tencent-cloud-cos**: 腾讯云对象存储 (文件管理、图片处理、智能搜索)
- **tencentcloud-lighthouse-skill**: 腾讯云轻量服务器 (实例管理、监控、防火墙、远程命令)

#### 搜索与信息获取 (2个)
- **tavily**: AI优化搜索 (需要TAVILY_API_KEY)

#### 文档处理 (3个)
- **tencent-docs**: 腾讯文档 (支持9种文档类型，智能文档为首选)
- **summarize**: 内容摘要 (支持网页/PDF/图片/音频/YouTube)
- **obsidian**: Obsidian知识库操作

#### 工具与实用程序 (7个)
- **github**: GitHub CLI操作
- **healthcheck**: 主机安全加固
- **tmux**: Tmux会话控制
- **video-frames**: 视频帧提取
- **weather**: 天气查询
- **Agent Browser**: 浏览器自动化
- **openai-whisper**: 语音转文字

#### 技能管理 (3个)
- **clawhub**: ClawHub CLI (搜索/安装/更新技能)
- **find-skills**: 技能发现 (优先skillhub，后备用clawhub)
